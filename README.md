<動作必要環境>
・Windows
・Python 3.X
・pycrypto

<暗号>
このプログラムには，ファイル送信機能とメッセージ送信機能がありますが，それは暗号化されません．
ただし，RSA暗号の2048bitの鍵を使用して，送信元の保障は行われます．この保証は，1回目のメッセージなどの受信時には適用されません．
送信元の保障方法は，悪意のある第3者がメッセージを偽装して送信してきた場合には，メッセージとファイルは正しく復号されないという形で行われます．

＜使用法＞
main.pyをpythonで起動
ダウンロードしたファイルはDownloadFilesフォルダに生成されます
